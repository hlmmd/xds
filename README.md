# xdsxt

## 环境配置

ubuntu 16.04

sudo apt install npm

sudo npm install express -gd

sudo npm install express-generator

创建项目

mkdir xds
cd xds

npm init 

sudo npm install -g express-generator

sudo  npm install -g express

express --version

npm install express --save

express -e test

安装 mysql

sudo apt install mysql-server
